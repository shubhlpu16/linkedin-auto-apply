This is a Chrome browser extension that automates job applications on LinkedIn. It uses LinkedIn's "Easy Apply" feature to streamline the application process.

üöÄ Installation Steps

  1. Step 1: Open Google Chrome browser
  2. Step 2: Navigate to:
    chrome://extensions/
  3. Step 3: Enable "Developer mode" using the toggle in the top-right corner
  4. Step 4: Click the "Load unpacked" button
  5. Step 5: Select this project folder
  6. Step 6: The extension should now appear in your extensions list
  7. Step 7: Pin the extension to your toolbar for easy access

üìñ How to Use

  1. Configure Profile: Click the extension icon and fill in your full name, email, phone number, and skills
  2. Save Settings: Click "Save Profile" to store your information
  3. Navigate to LinkedIn: Go to LinkedIn jobs and search for positions
  4. Start Auto Apply: Open the extension side panel and click "Start Auto Apply"
  5. Monitor Progress: Watch the applied and skipped counters in real-time
